# Seasonal Calendar — By Geography

This calendar drives contextual awareness for all agents. Sentinel uses it to explain performance spikes/dips. Lens uses it to tag seasonal creatives. Forge uses it to anchor copy in timely moments. Oracle uses it to recommend upcoming content windows.

---

## How to Use This Calendar

- **Forge:** When generating copy, check if a seasonal moment is within 2–4 weeks. If yes, offer a seasonal angle alongside the evergreen angle.
- **Lens:** Tag any creative that references a seasonal event with `H-EVENT` and the specific event code.
- **Oracle:** Flag upcoming windows in the pipeline status: "NAPLAN prep window opens in 3 weeks — no AU creatives in rotation."

## Australia Calendar

> Seasons are reversed: Summer = Dec–Feb, Winter = Jun–Aug.

### School Rhythm (4-term system)
| Term | Dates (approx) | Marketing Angle |
|---|---|---|
| Term 1 | Late Jan – Early Apr | New year, set up for NAPLAN |
| Term 1 Break | ~2 weeks April | Short break catch-up |
| Term 2 | Late Apr – Late Jun | NAPLAN month (May), mid-year push |
| Term 2 Break | ~2 weeks July | Winter break — "don't lose momentum" |
| Term 3 | Mid Jul – Late Sep | Selective school test prep |
| Term 3 Break | ~2 weeks October | Spring break |
| Term 4 | Mid Oct – Mid Dec | Year-end, report card season |
| Summer Break | Mid Dec – Late Jan | Longest break — summer learning loss angle |

|---|---|---|

## Calendar Maintenance Rules

1. **Never bundle festivals.** Each gets its own row, timing, and audience notation.
2. **Verify Islamic calendar dates annually** — they shift every year.
3. **Australian seasons are reversed.** Summer campaigns in AU run Dec–Feb, not Jun–Aug.
4. **India's academic year starts April** (in most states), not September.
5. **MEA has multiple overlapping school systems** — CBSE schools in UAE follow Indian calendar; MOE schools follow UAE calendar. Note which system the audience attends.
